# Спортивный Telegram‑бот (Fitness BOT)

Telegram‑бот для фиксации спортивной активности, отслеживания прогресса и достижения спортивных целей.

## Описание

Бот помогает пользователям:
* фиксировать спортивную активность (бег, велосипед, ходьба, плавание и т.д.);
* отслеживать динамику тренировок и прогресс;
* устанавливать цели и следить за их достижением;
* получать статистику по различным видам активности;
* настраивать напоминания о тренировках;
* экспортировать данные о тренировках.

## Команды

| Команда | Описание                                                                                          |
|-------|---------------------------------------------------------------------------------------------------|
| `/start` | Приветственное сообщение и краткая инструкция. Регистрация пользователя, если он ещё не зарегистрирован.|
| `/help` | Список доступных команд и инструкция по их использованию.|
| `/register` | Регистрация нового пользователя. Если пользователь уже зарегистрирован, бот сообщит об этом.|
| `/login` | Авторизация пользователя для доступа к его данным. Требуется повторная авторизация каждые 6 часов.|
| `/add_workout` | Добавление новой тренировки. Пример ввода: `13.01.2026 бег 5 км`.|
| `/add_goal` | Установка цели для тренировок. Пример: `13.03.2026 19.07.2026 Пробежать 50 км`.|
| `/view_workouts_group` | Просмотр истории тренировок без привязки к цели с группировкой.|
| `/view_workout_detail` | Просмотр детализированной информации по видам тренировок.|
| `/view_goals` | Просмотр целей и прогресса по ним.|
| `/reminder_off_achieved` | Отключение ежедневных уведомлений о просроченных и достигнутых целях.|
| `/workouts_no_goal_export_json` | Экспорт данных о тренировках без привязки к целям в формате JSON.|
| `/workouts_goal_export_json` | Экспорт данных о целевых тренировках в формате JSON.|
| `/cancel` | Сброс ввода последней команды.|

## Быстрый старт

1. Откройте Telegram и найдите бота.
2. Отправьте команду `/start`, чтобы получить приветственное сообщение и пройти регистрацию.
3. Используйте команду `/help`, чтобы ознакомиться со всеми возможностями бота.
4. Начните фиксировать свои тренировки с помощью команды `/add_workout`.
5. Устанавливайте цели через `/add_goal` и отслеживайте прогресс с `/view_goals`.

## Особенности

* **Удобный интерфейс:** на клавиатуре доступны две основные кнопки:
  * 💡 `/help` — список доступных команд;
  * 🆑 `/cancel` — сброс ввода последней команды.
* **Автоматические напоминания:** бот может напоминать о необходимости заниматься спортом.
* **Экспорт данных:** возможность выгружать историю тренировок в JSON для дальнейшего анализа.
* **Безопасность:** авторизация с ограничением активности (требуется повторный вход каждые 6 часов).
